Detailed Description

template<typename N>
class pan::detail::Graph< N >

Adapter around boost::adjacency_list.

Parameters
firsttemplate paramater. set::S to disallow multigraph
secondtemplate parameter. vec::S to access vertices using indeces
thirdtemplate parameter. undirectedS for undirected graphs.
forthtemplate parameter. The type to store as the vertex. Current configuration of the Class allows represents a simple, undirected graph. It allows self loop, however the self-edges are not created automatically.
Author
Hrachya Hakobyan

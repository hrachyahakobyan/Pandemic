template<>
bool pan::ActionHandler::validate ( const Move &  m) const

Validates Move action. A move action is valid if the following hold 1.The player exists.

  1. It is the player's turn.
  2. The player can make an action.
  3. The target city is not the same as the player's city.
  4. The target city is a neighbor of the player's city.

template<>
bool pan::ActionHandler::validate ( const DiscoverCure &  m) const

validates the discover cure action The action is valid if the following hold

  1. The initiating player exists.
  2. The initiating player may perform an action.
  3. The initiating player is located in a city that has a research station
  4. The initiating player has enough city cards of the same color as the disease he/she attempts to cure.
